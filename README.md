# 🛒 Mini Shopping Cart (React + Redux)

A minimal shopping cart application built using React and Redux Toolkit.

## 🚀 Features
- Add products to cart  
- Increase / Decrease quantity  
- Remove items  
- Automatic total price calculation  

## 🛠 Tech Stack
- React  
- Redux Toolkit  
- React Redux  

## ▶️ Run Locally

```bash
git clone <https://github.com/tapasisamal/mini-shopping-cart-redux>
cd mini-shopping-cart-redux
npm install
npm start
